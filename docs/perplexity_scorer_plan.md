# 한국어 퍼플렉서티 스코어러

> 직접 학습한 GPT-2 124M 모델을 한국어 텍스트 품질 평가 도구로 활용

---

## 목적

`ckpt_step200000.pt` 모델은 Q&A 능력이 없지만, 나무위키/한국어 Wikipedia로 학습했기 때문에
**한국어 문장의 자연스러움을 점수로 측정하는 용도**로는 바로 쓸 수 있다.

---

## 퍼플렉서티란?

```
PPL = exp(평균 cross-entropy loss)

낮을수록 → 모델이 "자연스럽다"고 판단
높을수록 → 모델이 "이상하다"고 판단
```

---

## 활용 예시

| 입력 문장 | 예상 PPL |
|-----------|----------|
| "인공지능은 데이터를 학습하여 패턴을 찾는다." | 낮음 (자연스러운 한국어) |
| "rlH faither gradient 조절 방법입니다." | 높음 (이상한 텍스트) |
| "NASA는 모든 사람들에게 알고 있다!" | 높음 (비문) |

---

## 구현 계획

1. `engine/transformer.py` 모델 로드
2. 문장 입력 → 토크나이즈 → loss 계산
3. `PPL = exp(loss)` 반환
4. 여러 문장 배치 처리 지원
