# LLM 수학 공식 학습 템플릿

```
나는 신입 ML/AI 엔지니어 면접을 준비 중이야.
아래 개념을 백지에 유도할 수 있을 때까지 도와줘.

[개념]: 
[공식]: 

다음 순서로 진행해줘:

1. 공식 없이, 이 개념이 왜 필요한지 한 문장으로 설명해줘
2. 수식이 왜 이렇게 생겼는지 직관적으로 유도해줘
3. ML/AI에서 어디에 쓰이는지 예시 2개만 줘
4. 연결된 다음 개념이 뭔지 알려줘 (내가 다음에 뭘 공부해야 하는지)

설명 끝나면 나한테 질문 하나 던져줘.
내 답변이 맞으면 다음으로 넘어가고, 틀리면 다시 설명해줘.
```

---

## 사용법

`[개념]`과 `[공식]` 두 줄만 바꾸면 어떤 공식이든 재사용 가능.

## 예시

```
[개념]: 내적 (Dot product)
[공식]: a·b = Σ(aᵢ·bᵢ)
```

```
[개념]: 소프트맥스 (Softmax)
[공식]: exp(xᵢ) / Σexp(xⱼ)
```

```
[개념]: Adam 옵티마이저
[공식]: m=β₁m+(1-β₁)∇, v=β₂v+(1-β₂)∇², w←w-lr·m̂/(√v̂+ε)
```
